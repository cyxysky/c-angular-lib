多维流程图 MultiDimensionalFlowchart

用于展示具有多个维度的流程图，支持主流程和子流程的可视化展示。

何时使用

当需要展示复杂的流程关系，特别是包含多个分支或多个层级的流程时，比如项目开发流程、工作流程、业务流程等需要展示层级关系的场景。

代码演示

最基本的流程图展示，配置数据中的steps定义流程步骤，stages定义流程阶段，通过nowStage指定当前所处阶段。 背景颜色可以自定义

通过配置stageArray可以自定义每个阶段的布局，以满足不同复杂度的流程图展示需求。

通过配置state可以设置流程的当前状态，支持active、hold、close三种状态。

组件支持自定义阶段的布局，通过stageArray属性配置多维数组，每个子数组表示一个阶段组，数组中的数字表示该阶段组中包含的阶段数量。
