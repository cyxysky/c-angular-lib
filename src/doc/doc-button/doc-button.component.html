Button 按钮

按钮是用户界面中最基本的交互元素，用于触发操作或事件。

何时使用

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

主要颜色 成功颜色 警告颜色 危险颜色 次要颜色 浅色颜色 深色颜色 灰色颜色 幽灵颜色

按钮有不同的颜色变体来表示不同的语义：

  • 主要颜色 (primary)：用于主要操作按钮
  • 成功颜色 (success)：表示成功或确认操作
  • 警告颜色 (warning)：提示用户谨慎操作
  • 危险颜色 (danger)：表示可能的危险操作
  • 次要颜色 (tertiary)：次要操作按钮
  • 浅色颜色 (light)：浅色背景按钮
  • 深色颜色 (dark)：深色背景按钮
  • 灰色颜色 (medium)：灰色背景按钮
  • 幽灵颜色 (ghost)：无背景按钮
默认按钮 虚线按钮 文字按钮 链接按钮

按钮有多种类型可供选择：

  • 默认按钮：标准按钮样式
  • 虚线按钮 (dashed)：使用虚线边框，常用于次要操作
  • 文字按钮 (text)：没有边框和背景，看起来像文本
  • 链接按钮 (link)：看起来像链接的按钮
默认形状 圆形

按钮可以有不同的形状：

  • 默认形状：标准矩形按钮
  • 圆形按钮 (circle)：圆形按钮，通常用于图标按钮
小按钮 默认尺寸 大按钮

按钮支持三种尺寸：

  • 小按钮 (small)：适用于空间有限的区域
  • 默认尺寸：标准按钮大小
  • 大按钮 (large)：需要强调或在移动设备上提供更大点击区域时使用
正常状态 禁用状态

按钮可以处于不同的状态：

  • 正常状态：可点击的按钮
  • 禁用状态 (disabled)：不可点击的按钮，通常用于表示操作当前不可用
