复选框 Checkbox

用于在一组可选项中进行多项选择。

何时使用

当用户需要从一组数据中选择多个选项时； 当用户需要表示开启/关闭两种状态时，可以使用单个复选框。

代码演示

当前值: {{ basicValues | json }}

最简单的用法，展示多个复选框选项。

当前值: {{ verticalValues | json }}

通过设置 direction="vertical" 属性使复选框垂直排列。

当前值: {{ colorValues | json }}

可以自定义复选框的选中颜色。

当前值: {{ disabledValues | json }}

可以禁用整个组件或单个选项。

当前值: {{ indeterminateValues | json }}

复选框支持半选状态，通常用于表示子选项部分选中的父级节点。
{{option.label}} - 自定义 (值: {{option.value}})

当前值: {{ templateValues | json }}

可以使用模板自定义复选框标签的显示方式。
