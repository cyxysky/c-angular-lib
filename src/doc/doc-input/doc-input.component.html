输入框 Input

基本的表单输入组件，用于接收用户输入的信息。

何时使用

需要用户输入文本内容时，可以使用输入框组件。适用于表单数据收集、搜索框、用户名密码等各类场景。

代码演示

当前值: {{ basicValue }}

最简单的用法，指定输入框的提示文字和默认值。
输入框有三种尺寸：大、默认和小，可以根据不同场景选择合适的尺寸。
可以添加前缀和后缀，用于在输入框前后添加文本或图标。
可以添加清除按钮，点击清除按钮可以一键清除输入框内容。
输入框可以设置为禁用或只读状态，禁用状态下不可编辑，只读状态下可以选择但不能修改。
输入框可以有不同的状态，用于表示错误或警告信息。
可以限制输入字符的最大数量，并显示字数统计。
无边框样式的输入框，适用于特定场景。
