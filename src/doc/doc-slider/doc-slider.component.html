滑动条 Slider

滑动型输入器，展示当前值和可选范围。

何时使用

当用户需要在数值区间内进行选择时； 当需要展示连续区间的值选择时； 当需要选择多个区间值时。

代码演示

当前值: {{ basicValue }}

最简单的滑动条，展示当前值和可拖拽的滑块。

当前值: {{ colorValue }}

可以自定义滑动条的轨道颜色。

当前值: {{ stepValue }}

设置 step 可以控制滑块的步长，默认为 1。

当前范围: [{{ rangeValue[0] }}, {{ rangeValue[1] }}]

设置 isRange 为 true 可以使用范围滑块，用于选择一个区间。

当前值: {{ marksValue }}

{{ mark.label }}
使用 marks 属性可以显示刻度标记。标记可自定义

当前值: {{ snapValue }}

设置 snapToMarks 为 true 可以让滑块只能对齐到刻度上。

当前值: {{ tooltipValue }}

通过 tipFormatter 函数可以自定义提示的显示格式。

当前值: {{ eventValue }}

上次改变: {{ lastChangedValue }}

可以监听滑块值的变化事件。
