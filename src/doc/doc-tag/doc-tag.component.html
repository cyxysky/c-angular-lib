Tag 标签

进行标记和分类的小标签。

何时使用

用于标记事物的属性和维度；进行分类。

代码演示

基础标签的用法，展示文本内容。
设置 closable 属性可以定义一个标签是否可以关闭。关闭标签时会触发 close 事件，从而可以在回调中做一些其他操作。
设置 checkable 属性可以定义一个标签是否可以选中。可以通过双向绑定 checked 属性控制标签的选中状态。

预设颜色

自定义颜色

我们提供了多种预设色彩的标签样式：primary、pink、red、orange、yellow、green、blue、purple等。
您也可以设置自定义颜色对象，通过指定背景色、文本色和边框色。
设置 tagBorder 属性为 false 可以创建无边框的标签。
设置 disabled 属性可以禁用一个标签，使其无法被点击、选中或关闭。
