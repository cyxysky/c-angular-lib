日期时间选择器 DateTimer

用于选择日期、时间或日期时间的输入组件。

何时使用

当用户需要输入一个日期，可以点击标准输入框，弹出日期面板进行选择。

代码演示

当前值: {{ getDisplayDate(basicValue) }}

最简单的用法，基础的日期选择器。

当前值: {{ selectTypeValue }}

日期时间选择器支持单选和范围选择两种类型。

当前值: {{ modeValue | json }}

当前模式: {{ currentMode }}

支持年、月份、季度、周、日、时间多种精度的选择。
日期时间选择器有三种尺寸：大、默认、小。

开始日期: {{ rangeValue.start | date }} / 结束日期: {{ rangeValue.end | date }}

日期时间选择器支持选择一个时间范围。
可以通过设置 disabledDate 和 disabledTime 来禁用一些特定的日期和时间。
可以为日期选择器添加状态（error、warning）以及设置无边框样式。
